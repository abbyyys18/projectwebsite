Section 1

User instructions/README

To start project, install scrapy and then run these files:

  • settings.py
  • middlewares.py
  • settings.py
  • items.py
  • indeed_spider.py
  • app.py

After running app.py, you can open the locally hosted web app and click the start button. This automatically activates the spider and crawls the web real time, so you might have to wait for a few minutes while it fetches the data.

Don't worry , I added some fun things on the app that will help you pass time!
